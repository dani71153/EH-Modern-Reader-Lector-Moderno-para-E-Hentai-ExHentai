# Release Notes

## v2.5.1 (2026-03-22)
### Added
- 新增 **hitomi.la** 站点完整支持：
  - 详情页（`*-{id}.html`）启动按钮
  - Reader 页（`/reader/{id}.html`）启动
  - 缩略图点击直达指定页

### Fixed
- 修复 nhentai / nhentai.xxx 在部分页面的按钮挂载与缩略图接管稳定性。
- 修复“阅读记忆 vs 显式跳页”优先级：
  - 详情页按钮启动：优先恢复阅读记忆
  - 缩略图点击启动：优先进入点击对应页
- 修复直接图片 URL 预取阶段的跨域兼容问题，减少 CORS 失败噪声。

### Changed
- 打包脚本已纳入 `hitomi.js`，确保发布包包含全部站点桥接能力。
- 清理根目录历史版本遗留说明文档，仓库结构更简洁。

## v2.5.0 (2026-03-22)
### Added
- 新增 **nhentai** 支持：
  - 支持 `/g/{id}/` 详情页按钮启动
  - 支持点击缩略图直接进入阅读器并跳转到对应页
  - 支持 `/g/{id}/{page}/` 从当前页启动
- 新增 `nhentai.js` 启动桥接脚本，复用现有现代阅读器内核

### Fixed
- 修复 nhentai 启动时偶发 `no gallery data`：改为页面上下文桥接（注入脚本 + `CustomEvent`）读取站点变量
- 增强 Gallery 模式兼容：支持直接图片 URL（jpg/png/gif/webp/avif）加载，不再强依赖 E-Hentai `/s/` 页面解析
- 增加启动防抖与可重试逻辑，避免重复点击时连续报错

## v2.4.9 (2026-01-23)
### 🐛 Bug Fixes
- **连续模式阅读记忆修复**：修复纵向/横向连续模式不保存阅读进度的问题，重进画廊现在能正确恢复到上次阅读位置
- **非虚拟纵向侧边留白**：调整滑块后立即重算所有图片高度，不再需要重进模式

### 📝 v2.3.5 ~ v2.4.9 完整更新摘要
自 v2.3.5 以来的重大更新：

#### 🚀 虚拟滚动
- 超过 200 页自动启用虚拟滚动，仅渲染可视区 ± 8 页
- 横向/纵向模式均支持，2000+ 页画廊流畅滚动
- 内存占用大幅降低（1996 页仅 ~20 DOM 元素）

#### 🎨 串珠式拖拽翻页
- 单页模式新增类似 JHenTai/iOS PageView 的拖拽效果
- 三滑块轨道设计，15% 阈值触发，280ms 缓动动画
- 鼠标拖拽同时支持所有连续模式

#### 🔧 稳定性修复
- 跳转位置稳定性大修（pendingJumpTarget 锚点机制）
- 模式切换页码保持、布局一致性
- 间距/侧边距设置实时生效
- 翻页尺寸闪跳修复（Image.decode 预解码）

## v2.3.4 (2025-11-14)
### Features
- 评论弹窗新增浮动“发评论”按钮：固定右下角，点击平滑滚动并聚焦输入区，避免长列表手动拉到底。
### UX / Improvements
- 弹窗打开后默认自动展开全部评论（移除“展开全部评论”按钮/链接交互），样式更贴近原站弹窗；悬浮按钮不遮挡内容，具备悬停提升与阴影渐变；关闭弹窗自动移除，保持 DOM 干净。
### Notes
- 若站点未来调整表单结构，仅需在 `gallery.js` 中更新选择器逻辑（查找 textarea/form），无需更改其它文件。

### Fixes
- 评论弹窗“展开全部评论”失效回画廊：现在拦截默认跳转并在弹窗内本地抓取 `?hc=1` 版本替换内容；修复展开后弹窗被自动关闭并滚动回原页面的问题（全局捕获 ?hc=1 / #cdiv 链接与鼠标事件，阻止导航）。
### Features
- 注入独立展开按钮（若原站缺失），状态指示：初始/加载中/已展开/失败；重复打开时保持已展开状态。
### Technical
- 使用事件委托统一拦截任何文本匹配“展开全部评论”的链接，同时在弹窗面板级别捕获潜在触发跳转的所有相关链接（?hc=1 / #cdiv），避免站点脚本导航导致弹窗消失；抓取完成后仅替换 `#cdiv` 内部，保持外层占位与关闭逻辑稳定。
### Notes
- 若后续站点改动评论完整加载参数（当前 `?hc=1`），仅需在 gallery.js 内更新 URL 逻辑，无需调整其它结构。

## v2.3.2 (2025-11-14)

### Changes
- 加强原站脚本阻断：注入阶段移除匹配 mpv 的脚本与样式链接，后续动态插入立即截获删除。
### Internal
- 统一错误关键字过滤（含 `offsetTop`）降低控制台噪声，聚焦扩展自身日志。
### Notes
- 若需再次启用“基于 Gallery 小图快速占位”模式，可在后续版本以设置开关形式恢复，当前版本优先保证稳定与一致视觉。
